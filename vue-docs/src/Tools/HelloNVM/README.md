##  概述

Manage multiple installations of node.js on a Windows computer.

我们在进行开发的使用往往需要切换不同的 node 环境进行，手动切换 node 的版本繁琐又容易出错使用 nvm （Node Version Management） 可以帮我们丝滑的管理你系统中的 node 环境。

## Features

- 操作简单
- 安装方便
