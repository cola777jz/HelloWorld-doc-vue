## 概述

:::danger

默认的 AList 只有很少的功能是为了避免百度网盘被和谐（限速）

如果你需要该功能请联系我我会为你提供账户

:::

ColaAList 是我通过开源项目 `AList` 搭建出来的一个网盘管理页面，它可以帮助你：

- 更直观的管理网盘中的文件
- 提供直链解析（无限速）下载资源 （为了防止接口被盗刷如果你需要该功能你可以联系我）
- 视频在线播放
- WebDav 挂载
- 通过页面元信息帮助你更好的管理网盘中的文件
